  • Monthly consolidated MANCOM reports

  • KPI reporting on performance and service health

Monitoring & Alerting

  • Fault identification, alerting and SOP-driven resolution via CMDB
  • ITOC event detection, filtering, validation and escalation
  • AFN Level 1 & 2 for network and application performance monitoring
  • DEM synthetic monitoring to catch issues before users notice

Incident & Problem Mgmt

  • End-to-end incident tracking and SLA management in ITOC
  • Problem identification, trending and aged analysis
  • IP conflict detection and escalation via IPAM events
  • Log collection, analysis and security incident reporting

Visualisation & Diagrams

  • Automated L2/L3 network diagrams updated weekly
  • VLAN, switch, router and subnet visualisation
  • FSO single-pane service dashboards
  • IPAM subnet and event dashboards

Data & Integration

  • CMDB as the structured data backbone — FCAPS model
  • Integration services: ITSM, SD-WAN, Orchestration, Workflow, APIs
  • FSO framework with ODBC, API Endpoint, Syslog, Meraki, WLC connectors
  • AIOps ETL pipelines in IPAM for automated data processing

Network & Flow

  • AFN: Core NG Flow, NetScout NG1, ThousandEyes for flow analysis
  • DEM: DNS, HTTP, cloud (AWS/Azure/GCP), SaaS, synthetic monitoring
  • IPAM: automated subnet discovery, VLAN mapping, duplicate IP detection
  • Drawing: automated topology mapping, MAC address and VLAN visualisation

Platform & Infrastructure

  • Hardware aaS: hardware/software installation, TAC case management
  • DR aaS: datastore replication, NMS switching, infrastructure recovery
  • EUC: endpoint configuration, patch management, IP asset treatment
  • Log: centralised syslog, retention policies, forensic archiving

Quick Reference

Key Terms Explained

Plain-language definitions for technical terms used across the platform.

CMDB
Configuration Management Database — the master record of all IT assets, configurations and relationships
IPAM
IP Address Management — tracks every IP address, subnet and VLAN to prevent conflicts and support planning
AFN
Application Flow Network — tools that measure, detect and investigate how network traffic and applications perform
DEM
Digital Experience Monitoring — monitors the experience of real users across networks, apps, cloud and SaaS
FSO
Full Stack Observability — a single dashboard pulling metrics from multiple tools into one unified view
EUC
End User Computing — management of workstations and end-user devices as trackable, managed IP assets
ITOC
IT Operations Centre — the team and tooling that manages events, incidents and problems in real time
FCAPS
Fault, Configuration, Accounting, Performance, Security — the five pillars of network management in the CMDB
DR aaS
Disaster Recovery as a Service — capability to restore monitoring infrastructure after a failure or outage
SOP
Standard Operating Procedure — documented steps teams follow for consistent, repeatable handling of events
ETL
Extract, Transform, Load — automated process that moves and enriches data between systems
CHMS
Combined managed solution grouping CMDB, Asset, EUC, Log, ITOC and Drawing into one package
